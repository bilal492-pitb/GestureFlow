## How to Use

1.  **Start the App**: Run `npm run dev` in the project directory.
2.  **Open Browser**: Go to `http://localhost:3000`.
3.  **Enable Camera**: Click the camera icon in the sidebar to start gesture detection.
4.  **Control PowerPoint**: Use the gestures above to control your presentation.

> [!TIP]
> Make sure you have PowerPoint installed on your system. The app uses PowerShell to communicate with PowerPoint via COM.

> [!IMPORTANT]
> If gestures are not being detected, ensure your hand is clearly visible to the camera and there is sufficient lighting.
